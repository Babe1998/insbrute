# Tnsbrute
Simple python script to bruteforce instagram using selenium webdriver

## Clone and change directory

```
git clone https://github.com/eternalodball/insbrute

cd insbrute
```

## Install requirements
```
sudo bash install.sh
```
## Run it !

```
python3 main.py
```
